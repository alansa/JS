ARRAYS O ARREGLOS O MATRICES

Las variables guardadas en un array en javascript no tienen que ser forzosamente 
del mismo tipo.

Hay dos formas de declarar un array en JavaScript

var nombreArray = ["valor1", "valor2", "valorN"];

var nombreArray = new Array ("valor1", "valor2", "valorN");

var nombreArray = [];

PROPIEDADES Y MÉTODOS DEL ARRAY IMPORTANTES

-Propiedad length: Longitud y número de elementos que tiene un Array

	nombreArray.length;

Se puede usar para obtener el último valor del Array si se desconoce el 
número de variables guardadas

	nombreArray(nombreArray.length]);

-Método push : Agrega un nuevo elemento al final del Array
 	
	nombreArray.push("valor a agregar");

	nombreArray.push("valor1 a agregar" , "valor2 a agregar","valorN a agregar");

-Método unshift : Agrega un nuevo elemento al inicio del Array

	nombreArray.unshift("valor a agregar");

	nombreArray.unshift("valor1 a agregar" , "valor2 a agregar","valorN a agregar");

-Método pop: Elimina un elemento al final del Array

	nombreArray.pop();

-Método shift: Elimina un elemento al inicio del Array

	nombreArray.shift();


PROGRAMACIÓN ORIENTADA A OBJETOS JS

ATRIBUTOS Y METODOS

NOMENCLATURA DEL PUNTO

Consiste en:

nombre del objeto  " . " despues su propiedad igual al valor;

Ejemplo Modificar el color y el ancho de un coche

renault.color="azul";
renault.ancho="2000";

Para que un objeto realice un metodo será asi

nombre del objeto " . " despues el metodo "()" ;

Ejemplo si queremos que el coche acelere y frene

renault.acelerar();
renault.frenar();

BOTONES EN JS

Modificar ancho de un boton, el color y entrar a un método del boton

boton.style.width="800px"
boton.style.backgroundColor = "red";
boton.focus();
